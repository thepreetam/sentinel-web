# SENTINEL — The First Machine-Vision Codec

Landing page for Sentinel Edge Gateway, a JEPA-based learned video codec for satellite backhaul optimization.

Built with Next.js 16, Tailwind CSS v4, and TypeScript.

## Development

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Push to GitHub and connect to Vercel — auto-detects Next.js.

## Sections

- Architecture — Dual-layer SVC (Base + Enhancement)
- Innovations — Semantic latent, temporal prediction, learned entropy
- Workflow — Encode → Predict → Residual → Deploy
- Performance — Benchmarks vs H.265
- Use Cases — Offshore, pipeline, island, substation ROI
- Deployment — Jetson Orin, T4/A100, Ryzen AI
- Design Partners — Gateway trial program
